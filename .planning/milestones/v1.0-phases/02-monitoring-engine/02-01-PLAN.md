    **backend/app/monitoring/event_matcher.py** — EventMatcher class:

    Constants (at module level):
    ```python
    TEAM_WEIGHT_HOME = 0.35
    TEAM_WEIGHT_AWAY = 0.35
    TIME_WEIGHT = 0.30
    CONFIDENCE_THRESHOLD = 0.90   # >= this triggers auto-action
    TIME_WINDOW_MINUTES = 15      # full score within ± 15 min
    MATCH_CACHE_TTL = 86400       # 24 hours
    ```

    `compute_confidence(px_home, px_away, px_start, sdio_home, sdio_away, sdio_start, px_sport, sdio_sport) -> float`:
    - Return 0.0 immediately if sports don't match (case-insensitive)
    - Normalize names: `s.lower().strip()`
    - `home_score = fuzz.token_sort_ratio(norm(px_home), norm(sdio_home)) / 100.0`
    - `away_score = fuzz.token_sort_ratio(norm(px_away), norm(sdio_away)) / 100.0`
    - Time score: 1.0 if delta <= 15 min, linear decay to 0.0 at 30 min, 0.0 beyond 30 min
    - Return weighted sum
    - Add note: `# Threshold calibration: adjust weights/threshold after observing real API data (see RESEARCH.md Pitfall 2)`

    `get_cached_match(redis_client, px_event_id: str) -> dict | None`:
    - Key: `match:px:{px_event_id}`
    - Returns parsed JSON dict or None

    `cache_match(redis_client, px_event_id: str, match: dict) -> None`:
    - `redis_client.setex(key, MATCH_CACHE_TTL, json.dumps(match))`

    `invalidate_match_cache(redis_client, px_event_id: str) -> None`:
    - `redis_client.delete(key)` — call when event's scheduled_start changes during polling

    `EventMatcher` class:
    - `__init__(self, redis_client)` — store redis_client
    - `find_best_match(self, px_event: dict, sdio_games: list[dict]) -> dict | None`:
      - Check cache first; if hit, return cached match dict
      - Compute confidence for every sdio_game in sdio_games
      - Find the game with highest confidence
      - If best >= CONFIDENCE_THRESHOLD: cache match, return {sdio_game_id, confidence, is_confirmed: True}
      - If 0 < best < CONFIDENCE_THRESHOLD: return {sdio_game_id, confidence, is_confirmed: False, is_flagged: True}
      - If no meaningful scores: return None
    - px_event dict keys expected: `px_event_id`, `sport`, `home_team`, `away_team`, `scheduled_start` (ISO string or datetime)
    - sdio_game dict keys expected: `sdio_game_id`, `sport`, `home_team`, `away_team`, `scheduled_start` (ISO string or datetime)
    - Parse ISO strings to datetime with timezone if not already datetime
