    `get_expected_px_status(sdio_status: str) -> str | None`:
    - If sdio_status in FLAG_ONLY_STATUSES: return None (no auto-action)

    - Log a warning via structlog if sdio_status is not recognized (not in enum or mapping)

    `is_mismatch(px_status: str, sdio_status: str) -> bool`:
    - expected = get_expected_px_status(sdio_status)
    - If expected is None: return False (flag-only statuses are not auto-correctable mismatches)
    - Return px_status != expected (case-sensitive; ProphetX enum values must be confirmed)

    `is_flag_only(sdio_status: str) -> bool`:
    - Return sdio_status in FLAG_ONLY_STATUSES

    - If not found: return Decimal("0") with log warning "no_default_liquidity_threshold_configured"
